思路

假设 f(x)代表爬第x楼的方法数，则明显可知，f(0) = 1， f(1) = 1，f(2) = 2，而对于f(3)的话可以选择从一楼爬两个台阶或从二楼爬一个台阶，则f(3) = f(2) + f(1)，由此可以推广到f(x) = f(x- 1) + f(x - 2)

则可以设置三个变量a1，a2，a3，每次爬楼则将a2赋值给a1，将a3赋值给a2，将a1 + a2赋值给a3
